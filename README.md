# ChainGenieThis is a repo for managing machine readable artifacts for the [ChainGenie APIs](http://api.chaingenie.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [ChainGenie APIs](http://api.chaingenie.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://api.chaingenie.com)- [Developer](http://api.chaingenie.com)- [Twitter](https://twitter.com/ChainGenie)- [GitHub](https://github.com/ChainGenie)The [APIs.json](https://github.com/api-evangelist/chaingenie/blob/master/apis.json) index provides links to all the relevant links for the ChainGenie APIs, including the OpenAPI and Postman if they are present.
